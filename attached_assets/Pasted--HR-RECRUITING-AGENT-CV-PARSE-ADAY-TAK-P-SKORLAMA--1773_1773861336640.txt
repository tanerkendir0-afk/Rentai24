# HR & RECRUITING AGENT — CV PARSE + ADAY TAKİP + SKORLAMA

# ============================================================

# Bu talimatı Replit Agent’a olduğu gibi yapıştır.

# ============================================================

HR & Recruiting Agent’ı (hr-recruiting) gerçek İK operasyonları yapabilecek şekilde
güçlendirmek istiyorum. Şu an sadece 4 temel tool var: create_job_posting, screen_resume,
create_interview_kit, send_candidate_email. Bunlar metin bazlı — gerçek dosya okuma,
aday veritabanı, skorlama sistemi YOK.

KOBİ’ler için en değerli: “CV’leri yükle, otomatik skorla, en uygun adayı bul, mülakat planla.”

```sql
-- İlanlar
CREATE TABLE IF NOT EXISTS job_postings (
  id SERIAL PRIMARY KEY,
  user_id INTEGER NOT NULL REFERENCES users(id),
  title VARCHAR(300) NOT NULL,
  department VARCHAR(100),
  employment_type VARCHAR(50),      -- full-time, part-time, contract, intern, remote
  description TEXT,
  requirements TEXT,                 -- JSON array veya virgülle ayrılmış

  salary_min INTEGER,
  salary_max INTEGER,
  currency VARCHAR(10) DEFAULT 'TRY',
  location VARCHAR(200),
  status VARCHAR(20) DEFAULT 'active',  -- active, paused, closed, filled
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Adaylar
CREATE TABLE IF NOT EXISTS candidates (
  id SERIAL PRIMARY KEY,
  user_id INTEGER NOT NULL REFERENCES users(id),
  full_name VARCHAR(300) NOT NULL,
  email VARCHAR(300),
  phone VARCHAR(50),
  linkedin_url VARCHAR(500),
  current_title VARCHAR(300),
  current_company VARCHAR(300),
  experience_years INTEGER,
  skills JSONB,                      -- ["Python", "Excel", "Satış", ...]
  education TEXT,
  location VARCHAR(200),
  salary_expectation INTEGER,
  cv_file_path TEXT,                 -- yüklenen CV dosya yolu
  cv_text TEXT,                      -- parse edilmiş CV metin içeriği
  notes TEXT,
  source VARCHAR(50),                -- linkedin, kariyer.net, referral, direct
  created_at TIMESTAMP DEFAULT NOW()
);

-- Aday-İlan eşleştirme ve süreç takibi
CREATE TABLE IF NOT EXISTS applications (
  id SERIAL PRIMARY KEY,
  candidate_id INTEGER NOT NULL REFERENCES candidates(id),
  job_posting_id INTEGER NOT NULL REFERENCES job_postings(id),
  status VARCHAR(30) DEFAULT 'new',  -- new, screening, interview_scheduled, interviewed, offer, hired, rejected
  fit_score INTEGER,                 -- 0-100 otomatik skorlama
  screening_notes TEXT,
  interview_date TIMESTAMP,
  interview_notes TEXT,
  rejection_reason TEXT,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW(),
  UNIQUE(candidate_id, job_posting_id)
);
```

### 2. BACKEND — CV Parse Servisi

`server/services/cvParserService.ts` oluştur:

```typescript
// PDF ve DOCX formatındaki CV'leri parse eder
// Bağımlılıklar: pdf-parse, mammoth (docx→html), veya pdfjs-dist

import pdfParse from 'pdf-parse';
import mammoth from 'mammoth';
import fs from 'fs';

interface ParsedCV {
  fullText: string;
  sections: {
    contactInfo: {
      name?: string;
      email?: string;
      phone?: string;
      linkedin?: string;
      location?: string;
    };
    summary?: string;
    experience: {
      title: string;
      company: string;
      duration: string;
      description: string;
    }[];
    education: {
      degree: string;
      institution: string;
      year: string;
    }[];
    skills: string[];
    languages: string[];
    certifications: string[];
  };
}

// PDF'den metin çıkar
async function parsePDF(filePath: string): Promise<string> {
  const buffer = fs.readFileSync(filePath);
  const data = await pdfParse(buffer);
  return data.text;
}

// DOCX'den metin çıkar
async function parseDOCX(filePath: string): Promise<string> {
  const buffer = fs.readFileSync(filePath);
  const result = await mammoth.extractRawText({ buffer });
  return result.value;
}

// CV metnini yapılandır (LLM ile veya regex ile)
// Basit regex-based extraction + LLM enrichment
async function structureCV(rawText: string): Promise<ParsedCV> {
  // Email regex
  const emailMatch = rawText.match(/[\w.-]+@[\w.-]+\.\w+/);
  // Phone regex (TR formatları)
  const phoneMatch = rawText.match(/(\+90|0)\s*\d{3}\s*\d{3}\s*\d{2}\s*\d{2}/);
  // LinkedIn
  const linkedinMatch = rawText.match(/linkedin\.com\/in\/[\w-]+/);

  // Skill extraction — yaygın skill keyword'leri
  const commonSkills = [
    "Python", "JavaScript", "TypeScript", "React", "Node.js", "SQL", "Excel",
    "AutoCAD", "SAP", "Photoshop", "Figma", "AWS", "Docker", "Git",
    "Satış", "Pazarlama", "Muhasebe", "İnsan Kaynakları", "Proje Yönetimi",
    "Müşteri İlişkileri", "Liderlik", "İletişim", "Takım Çalışması",
    "İngilizce", "Almanca", "Fransızca", "İspanyolca",
    "SPSS", "Power BI", "Tableau", "Google Analytics",
    "C#", ".NET", "Java", "PHP", "Flutter", "Swift", "Kotlin",
  ];
  const foundSkills = commonSkills.filter(s =>
    rawText.toLowerCase().includes(s.toLowerCase())
  );

  return {
    fullText: rawText,
    sections: {
      contactInfo: {
        email: emailMatch?.[0],
        phone: phoneMatch?.[0],
        linkedin: linkedinMatch?.[0],
      },
      skills: foundSkills,
      experience: [],  // LLM ile daha detaylı çıkarılabilir
      education: [],
      languages: [],
      certifications: [],
    },
  };
}

// CV-İlan uyum skoru hesapla
function calculateFitScore(cv: ParsedCV, jobRequirements: string[]): number {
  const cvSkills = cv.sections.skills.map(s => s.toLowerCase());
  const reqSkills = jobRequirements.map(s => s.toLowerCase().trim());

  let matchCount = 0;
  for (const req of reqSkills) {
    if (cvSkills.some(s => s.includes(req) || req.includes(s))) {
      matchCount++;
    }
  }

  // Temel skor: eşleşen skill yüzdesi
  const skillScore = reqSkills.length > 0 ? (matchCount / reqSkills.length) * 100 : 50;
  return Math.round(Math.min(skillScore, 100));
}
```

Bağımlılıklar: `npm install pdf-parse mammoth`

### 3. BACKEND — CV Yükleme API

```
POST   /api/hr/upload-cv           — CV yükle (PDF/DOCX, max 5MB)
GET    /api/hr/candidates           — Aday listesi
GET    /api/hr/candidates/:id       — Aday detayı (CV metin dahil)
GET    /api/hr/job-postings         — İlan listesi
GET    /api/hr/applications         — Başvuru takip
PUT    /api/hr/applications/:id     — Başvuru durumu güncelle
```

CV upload sonrası otomatik:

1. Dosyayı kaydet (uploads/hr/)
1. PDF/DOCX parse et → metin çıkar
1. Yapılandırılmış veri çıkar (isim, email, skills)
1. candidates tablosuna kaydet
1. Aktif ilanlarla otomatik eşleştir ve skorla

### 7. SYSTEM PROMPT GÜNCELLEMESİ

HR agent’ın system prompt’una ekle:

```
## İnsan Kaynakları Yetenekleri

Sen tam donanımlı bir İK asistanısın. Aday takip sistemi (ATS), CV analizi,
skorlama ve işe alım süreç yönetimi yapabilirsin.

### CV yüklendiğinde:
1. upload_cv ile adayı kaydet
2. Aktif ilanlar varsa otomatik score_candidate çağır
3. Uyum skoru ve güçlü/zayıf yönleri belirt
4. Önerilen aksiyon sun (mülakat planla, reddet, havuzda tut)

### İşe alım akışı:
- "Yazılım geliştirici arıyorum" → create_job_posting
- "Bu CV'yi değerlendir" → upload_cv + score_candidate
- "Tüm adayları sırala" → bulk_score_candidates
- "En uygun 3 adayı göster" → list_candidates (sort: score, limit: 3)
- "Ali ile mülakat planla" → schedule_interview + send_candidate_email
- "Ayşe'yi reddet" → generate_rejection_email + update_application_status
- "Mehmet'e teklif hazırla" → generate_offer_letter
- "Pipeline durumu?" → hiring_pipeline_summary

### Skorlama kriterleri:
- Skill eşleşme: %50 ağırlık (ilandaki required skills vs CV skills)
- Deneyim yılı: %20 ağırlık
- Eğitim uyumu: %15 ağırlık
- Ek faktörler: %15 (sertifikalar, dil, lokasyon)

### Profesyonel iletişim:
- Ret email'leri empatik ve yapıcı olmalı
- Teklif mektupları profesyonel ve detaylı olmalı
- Mülakat davetleri tüm bilgileri (tarih, saat, yer/link, hazırlık notları) içermeli
- Türkçe ve İngilizce şablon desteği

### KVKK uyarısı:
- Aday verilerini gizli tut
- Sadece yetkili kullanıcıya göster (user_id kontrolü)
- CV'ler 90 gün sonra otomatik anonimleştir (isteğe bağlı)
```

### 9. GÜVENLİK

- CV dosyaları max 5MB
- Sadece .pdf, .docx, .doc kabul et
- Her kullanıcı sadece kendi adaylarını görebilir
- KVKK uyumlu: aday rızası olmadan 3. tarafla paylaşma
- CV metin içeriğinde XSS/injection koruması
