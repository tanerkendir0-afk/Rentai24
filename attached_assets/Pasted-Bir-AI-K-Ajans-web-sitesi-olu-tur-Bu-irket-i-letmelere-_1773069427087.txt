Bir "AI İK Ajansı" web sitesi oluştur. Bu şirket, işletmelere eğitimi tamamlanmış hazır AI ajanları kiralıyor. Geleneksel İK ajansları nasıl şirketlere insan çalışan yerleştiriyorsa, biz de aynı modelle AI çalışanlar (ajanlar) kiralıyoruz.

Teknoloji: Next.js + Tailwind CSS + TypeScript. Modern, profesyonel, koyu tema (dark mode) ağırlıklı tasarım. Türkçe dil desteği.

## AI MODEL ENTEGRASYONU İÇİN

`/api/chat` endpoint'indeki TODO kısmını kendi API'nizle değiştirmeniz yeterli. Örnek yapı:

```typescript
// /api/chat/route.ts
export async function POST(req: Request) {
  const { message, agentType } = await req.json();

  const response = await fetch('SİZİN_AI_API_URL', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${process.env.AI_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      messages: [{ role: 'user', content: message }],
      agent_type: agentType,
    }),
  });

  const data = await response.json();
  return Response.json({ reply: data.response });
}
